Projects

Below are a few of the key projects I have created. These projects are only dummy pages created for the purpose of showcasing my skills and my different design capabilities.

Nexter

Nexter is a Luxurious Real Estate company's website which showcases a slick elegant. It is built using only HTML and CSS. It also utilizes the grid design basis to its fullest extent. It includes a horizontal menu navigation bar on the left of the page and also contains a beatiful image gallery in the bottom of the page. The page is 100% responsive to any screen size (phones, tablets, desktops, etc.).

Forkify

Forkify is a Food Recipes application which utilizes the capabilities of Javascript. Recipes data is derived from a custom API (Application Programming Interface). It provides the user basic functionality like searching for recipes, bookmarking favorite recipes, and adding recipes. The user's data is based on the browser's local storage of the session to enhance and customize the user's experience during the visit to the website.

NaTours (Static)

Natours is an Adevnture Booking company's website. It displays the content in beautifully designed sections with advanced display properties like overlaying images, front and back facing cards, a testimonials section, a call-to-action form with custom radio buttons, and a comprehensive footer. Another key component on this page is the expanding menu in the top-right of the screen. The page is 100% responsive to any screen size (phones, tablets, desktops, etc.).

NaTours (Dynamic)

This application is the dynamic version of the Natours website built using Node.js technology. It involves an API which relies on data pulled from a noSQL cloud database. It features an interactive map and provides complete management of the tours(products), admin and user accounts, and stripe payments/checkout.
