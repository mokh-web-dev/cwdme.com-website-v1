Nexter (CSS Grid)

Nesxter is a Luxurious Real Estate company's website which showcases a slick and an elegant website design. It is build using only HTML and CSS. It also utilizes the grid design basis to its fullest extent. It includes a horizontal menu navigation bar on the left of the page and also contains a beatiful image gallery in the bottom of the page. The page is 100% responsive to any screen size (phones, tablets, desktops, etc.).

Trillo (CSS Flexbox)

Trillo is a Holiday Booking company's website/application which utilizes the capabilities of flexible boxes to its fullest. It includes a menu on the left side of the page with some nice animations on hover, uniformly constructed components like images and links, and a search bar in the top of the page. The page is 100% responsive to any screen size (phones, tablets, desktops, etc.).

Forkify (Javascript)

Forkify is a recipe application which utilizes the capabilities of Javascript. The data included is derived from a custom API (Application Programming Interface). It provides the user basic functionality like searching for recipes, bookmarking favorite recipes, and adding recipes. The user's data is based on the browser's local storage of the session to enhance and customize the user's experience during the visit to the website.
